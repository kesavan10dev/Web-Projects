Today_List

Today List is a responsive daily task management app built with Vue.js and Tailwind CSS, designed to help users organize and prioritize their day efficiently. It allows users to add, edit, delete, and mark tasks as complete, with support for due dates, filtering by status, and search functionality. The app features a clean, mobile-friendly UI using Tailwind’s utility classes and stores tasks in the browser using localStorage for persistence. Through this project, I gained hands-on experience with Vue 3’s Composition API, reactive state handling, dynamic rendering, and responsive design principles, while building a focused and user-friendly productivity tool.
